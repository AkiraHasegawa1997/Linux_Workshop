	ディレクトリ操作
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
‣ pwd
現在いる場所 (カレントディレクトリ) を表示する．
使用例 : 現在の場所を表示

	 $ pwd 

‣ ls
ディレクトリ（フォルダ）内にどのようなファイルがあるかを見るときに使います．
使用例 : 現在のディレクトリ内にあるファイル・ディレクトリ一覧を表示

	 $ ls 

‣ cd
他のディレクトリに移動するときに使います．
使用例 : ディレクトリ ex01 に移動

	 $ cd ex01 

1つ上のディレクトリと現在のディレクトリにはそれぞれ特別に .. と . が割り当てられています．
使用例 : 1つ上のディレクトリに移動

	 $ cd .. 

‣ mkdir
新しいディレクトリを作りたいときに使います．
使用例 : 新しいディレクトリ new_dir を作成

	 $ mkdir new_dir 

‣ rm -r
ディレクトリを削除するときに使います．
 一度間違えて消してしまうと復元することはできません． 
通常，警告文は出ないので注意して使いましょう．
使用例 : 不要なディレクトリ dir を削除

	 $ rm -r dir 

‣ mv
ディレクトリの場所を移動したり，名前を変更するときに使います．
 mv 移動対象 移動先 という形式で使用します．
使用例1 : ディレクトリ target_dir を1つ上の階層へ移動 

	 $ mv target_dir ../ 

使用例2 : ディレクトリ target_dir の名前を rename_dir に変更

	 $ mv target_dir rename_dir 

‣ cp -r
ディレクトリをコピーするときに使います．
 cp -r コピー元 コピーしたディレクトリにつける名前 という形式で使用します．
使用例 : ディレクトリ target_dir をコピーして copied_dir を作成

	 $ cp -r target_dir copied_dir

	実践演習
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
1.  ls を実行した後に，ディレクトリ mydir を作成し，再度 ls コマンドで確認してみましょう．

2.  pwd を実行した後に，1で作成した mydir に移動し，再度 pwd を実行してみましょう．

3.  mydir の名前を好きな名前に変更してみましょう．

4. 作成したディレクトリを削除してみましょう．

5.  ex01 の1つ上の階層にある ex02 へ移動し，次の演習に進みましょう．
