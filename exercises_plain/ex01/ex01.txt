	ディレクトリ操作
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
‣ pwd
現在いる場所 (カレントディレクトリ) を表示する．
使用例 : 現在の場所を表示

	$ pwd

‣ ls
ディレクトリ（フォルダ）内にどのようなファイルがあるかを見るときに使います．
使用例 : 現在のディレクトリ内にあるファイル・ディレクトリ一覧を表示

	$ ls

‣ cd
他のディレクトリに移動するときに使います．
使用例 : ディレクトリex01に移動

	$ cd ex01

1つ上のディレクトリと現在のディレクトリにはそれぞれ特別に..と.が割り当てられています．
使用例 : 1つ上のディレクトリに移動

	$ cd ..

‣ mkdir
新しいディレクトリを作りたいときに使います．
使用例 : 新しいディレクトリnew_dirを作成

	$ mkdir new_dir

‣ rm -r
ディレクトリを削除するときに使います．
一度間違えて消してしまうと復元することはできません．
通常，警告文は出ないので注意して使いましょう．
使用例 : 不要なディレクトリdirを削除

	$ rm -r dir

‣ mv
ディレクトリの場所を移動したり，名前を変更するときに使います．
mv 移動対象 移動先という形式で使用します．
使用例1 : ディレクトリtarget_dirを1つ上の階層へ移動 

	$ mv target_dir ../

使用例2 : ディレクトリtarget_dirの名前をrename_dirに変更

	$ mv target_dir rename_dir

‣ cp -r
ディレクトリをコピーするときに使います．
cp -r コピー元 コピーしたディレクトリにつける名前という形式で使用します．
使用例 : ディレクトリtarget_dirをコピーしてcopied_dirを作成

	$ cp -r target_dir copied_dir

4. 作成したディレクトリを削除してみましょう．
